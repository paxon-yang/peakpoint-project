# UI_BASELINE_V1.md
> CCSA项目管理 UI 封版基线（v1.0）  
> 封版时间：2026-04-14  
> 目标：后续所有界面迭代保持统一风格、统一节奏、统一交互反馈。

## 1. 视觉方向

- 风格：Notion Inspired（暖白、轻边框、低强度阴影）
- 气质：企业内网可读性优先、克制、稳定
- 禁止：高饱和多色、重阴影、夸张动画

## 2. 设计 Token（源于 `src/styles.css`）

- 背景：`--notion-bg: #ffffff`、`--notion-bg-warm: #f6f5f4`
- 文本：`--notion-text: rgba(0,0,0,0.95)`、`--notion-text-secondary: #615d59`
- 主色：`--notion-blue: #0075de`、激活 `#005bab`
- 边框：`1px solid rgba(0,0,0,0.1)`
- 圆角：`4px / 8px / 12px`
- 阴影：Notion 低透明多层阴影（已写入样式变量）

## 3. 布局基线

- 页面：头部 + 内容区（`content-wrap`）
- 主工作区：左右分栏（任务表 48% / 甘特图 52%）
- 面板：统一 `panel-card`（白底、轻边框、轻阴影）
- 移动端：1200px 以下改为单列堆叠

## 4. 组件规范

- 按钮：主按钮蓝底白字，次按钮低对比灰底
- 输入框：32px 高度，focus 使用蓝色 ring
- 表格：固定列宽（`TaskTable colgroup`），行内省略，禁止宽度抖动
- 标签：统一胶囊样式（12px、圆角 9999）
- 抽屉：右侧滑出，16px 内边距，标题 22px

## 5. 甘特图皮肤（第四/五轮收官）

- 包裹类：`.notion-gantt-theme`
- 时间头：暖灰背景 + 次级文本色
- 网格线：`rgba(0,0,0,0.06)`
- today：低透明蓝底
- 任务条：圆角、低饱和、高可读标签
- 依赖线：克制线宽与颜色

## 6. 可改与不可改

可改（不需改基线文档）：
- 文案
- 默认数据
- 业务字段和交互逻辑

需先更新本基线再改：
- 主色与中性色体系
- 主布局比例
- 按钮/输入/标签的全局尺寸
- 甘特图主题样式

## 7. 验收清单（封版）

- 中文文案无乱码
- 左表右图布局稳定
- 任务表列宽稳定，不跳动
- 甘特图日/周/月可切换
- 拖拽、进度、依赖线可用
- localStorage 持久化正常
